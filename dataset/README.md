# Dataset

## Overview

This dataset contains raw data collected during the reconnaissance phase of the Red Sentinel project, focusing on XSS (Cross-Site Scripting) vulnerabilities and payloads.

## Data Sources

### awesomeXss

Collection of XSS attack vectors and test cases gathered from security research and vulnerability databases.

**Repository**: https://github.com/s0md3v/AwesomeXSS

### payloadsAllTheThings

Comprehensive payload repository sourced from community-driven security resources and penetration testing frameworks.

**Repository**: https://github.com/swisskyrepo/PayloadsAllTheThings

### xssgai

XSS payload data collected through automated analysis and machine learning-based vulnerability detection.

**Repository**: https://github.com/AnonKryptiQuz/XSSGAI

## Data Insights

- **Coverage**: Multiple XSS attack vectors including DOM, stored, and reflected variants
- **Format**: Raw payloads and attack patterns in structured formats
- **Use Case**: Testing and validation of XSS detection mechanisms
- **Source Quality**: Data aggregated from established security research communities and open-source projects

## Note

This data is intended for authorized security research and defensive testing purposes only.
Unauthorized use of this data for malicious activities is strictly prohibited. Always ensure compliance with legal and ethical guidelines when utilizing this dataset.
